SUMMARY

As a Project Showcase for Bloc, I redesigned Beverlycastleacademy.org with a responsive website and new logo.
This project included: updated copy, a new color palette, updated typography and a reorganized sitemap for a better user experience.

THE CHALLENGE

Product Showcase Goal: Design and build a responsive web site to promote a product.
Beverly Castle Academy's redesign challenged me and reinforced the skills I learned in during Bloc.

UI DESIGN

Creating the Logo

This logo includes a castle and clover icon to represent the Irish community of Beverly.
The color palette I used is fun and youthful, but also takes on an academic look when paired with the Playfair Regular type.

Prototyping

I started sketching rough ideas of my prototype in my notebook, and then created my Version 1. Next, I drafted a version in Balsamiq and analyzed the designs with my Bloc mentor. I then used Sketch to create high-fidelity wireframes. Adding my updated logo, color scheme and font pairings made my wireframes come to life! The last image shows how the design appeared in Sketch.

User Testing

I used Marvel to contruct and test my interactive prototype. This was my first time using Marvel, and I had some difficulty figuring out to how create interactivity on a responsive website. However, it is still a work in progress! View my project on Marvel.

RESULTS & CONCLUSION

I had great feedback from my mentor on this redesign. I love the color template and fonts, and the flow of the new site. I will continue building on the skills learned from this project. View this project on my Github pages.

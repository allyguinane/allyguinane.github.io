SUMMARY

Treasurely is SaaS web application that allows people to collect and store information in a single place.
The goal: design a web application for users to create documents, store images, bookmark links to websites, and collaborate with other users in the application.

THE CHALLENGE

Treasurely was the first big project I worked on in the UX/UI Design Class. The challenge was to create a fresh note-keeping product that was different from competitors. I was introduced to, and used the concepts below:

  • User personas
  • User stories
  • User flows
  • Building sitemaps
  • Lo-fidelity wireframes
  • Responsive wireframes
  • High-fidelity mockups
  • Logo / branding
  • Style guides
  • Interactive prototypes
  • HTML/CSS development
  • Usability Testing

UI DESIGN

Creating the Logo

Treasurely’s logo shows a key, constructed from parts of the typeface I used for this logo. Tresurely’s motto is “Unlock Your Thoughts." I used a jewel-toned color palette to match the idea of finding treasures.

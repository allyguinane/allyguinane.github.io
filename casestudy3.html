Treasurely - A Bloc Project

  • Responsive Web Design
  • User Research & UI Design
  • Low Fidelity and High Fidelity Wireframes
  • Interactive Prototyping

SUMMARY

Treasurely is SaaS web application that allows people to collect and store information in a single place.
The goal: Design a web application for users to create documents, store images, bookmark links to websites, and collaborate with other users within the application.

THE CHALLENGE

Treasurely was the first project I worked on in the UX/UI design course.
The challenge: Create a fresh note-keeping product that was different from competitors.
I was introduced to, and used the following concepts in this project:

  • User personas, user stories and user flows
  • Sitemap construction and decision making
  • Low-fidelity and high-fidelity wireframes
  • HTML & CSS development and styling
  • Logo creation, branding
  • Interactive prototypes
  • Usability testing

UI DESIGN

Creating the Logo

Treasurely’s logo is a skeleton key, constructed from broken down shapes of the Roboto Slab typeface.
Treasurely’s motto is “Unlock Your Thoughts." I used a jewel-toned color palette to match the idea of finding treasures.

SOLUTION

User Personas

For this project, I was introduced to User Personas. I gathered information from three different potential users to assess their motivations, goals and frustrations with using similar products. This research allowed me to build a better note-taking product from the beginning stages.

User Flows
Creating Site Maps

Using Draw.io, I created a simple site map for the user. I designed a sign in and sign up process, a process to reset your password, and the whole experience of using the main dashboard in the product.

Prototyping

Using Balsamiq, I created several different versions of my low fidelity wireframes. After choosing a version that worked best for the user, I designed the high fidelity wireframes to include more detail using Adobe Illustrator and Photoshop. Finally, I added my color scheme.

User Testing

InVision was the base to contructing and testing my interactive prototype. After adding a few hotspots to the buttons for navigation, I did some user testing by watching my friends work through the site. I also viewed user testing videos of my product using UserTesting.com. Check out Treasurely on InVision.

RESULTS & CONCLUSION

Treasurely was a successful first project. I learned valuable beginner skills in Github, HTML, CSS, and learned to navigate user flows, site maps and user testing. View the project on my Github.

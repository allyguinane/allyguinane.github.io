Treasurely - A Bloc Project

  • Responsive Web Design
  • User Research & UI Design
  • Low Fidelity and High Fidelity Wireframes
  • Interactive Prototyping

SUMMARY

Treasurely is SaaS web application that allows people to collect and store information in a single place.
The goal: Design a web application for users to create documents, store images, bookmark links to websites, and collaborate with other users in the application.

THE CHALLENGE

Treasurely was the first big project I worked on in the UX/UI Design Class.
The challenge: Create a fresh note-keeping product that was different from competitors.
I was introduced to, and used the following concepts in this project:

  • User personas, user stories and user flows
  • Sitemap construction and decision making
  • Low-fidelity and high-fidelity wireframes
  • HTML & CSS development and styling
  • Logo creation, branding
  • Interactive prototypes
  • Usability testing

UI DESIGN

Creating the Logo

Treasurely’s logo shows a key, constructed from parts of the typeface I used for this logo.
Tresurely’s motto is “Unlock Your Thoughts." I used a jewel-toned color palette to match the idea of finding treasures.

SOLUTION

Research and User Testing
Writing User Personas, User Flows, Site Maps, Creating Protypes and learning to do User Testing.

User Personas

As a part of this project, I was introduced to User Persona's. I gathered information from three different potential users to assess their motivations, goals and frustrations with using similar products. This research allowed me to build a better note-taking product from the beginning stages.

User Flows
Creating Site Maps

Using Draw.io, I created a simple site map for the user.

Prototyping

I used Balsamiq to protoype my low fidelity wireframes. I designed the high fidelity wireframes using Adobe Illustrator and Photoshop.

User Testing

I used InVision to contruct and test my interactive prototype. Check out Treasurely on InVision.

RESULTS & CONCLUSION

Treasurely was a successful first project as I learned Github, used HTML and CSS, and navigated User Flows, Site Maps and User Testing. View the project on my Github.
